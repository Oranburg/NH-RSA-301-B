# N.H. RSA 301-B: Decentralized Autonomous Organizations

Full text of **N.H. Rev. Stat. Ann. ch. 301-B**, New Hampshire's Decentralized Autonomous Organization Act, with a navigable web display.

## The statute

RSA 301-B establishes DAOs as recognized legal entities in New Hampshire. Enacted through HB 645 (2024, ch. 263), the statute provides:

- **Legal personality** and limited liability for DAOs (sections 7, 11)
- A **registration framework** via the Secretary of State (section 12)
- The **UNH InterOperability Laboratory** as registry administrator (section 5, XXXVI)
- A mandate to build the registry **as a blockchain system** (section 14, VI)
- Governance through bylaws with a "code is law" policy principle (sections 3, 16)
- Rules for **contentious forks** and DAO restructuring (sections 25, 26)

## The site

A single-page statute reader built with vanilla HTML/CSS/JS and [marked.js](https://marked.js.org/). Features:

- Sidebar table of contents with scroll spy
- Dark/light theme toggle
- Section-level anchor links (click any heading to copy)
- Mobile responsive
- Downloadable Markdown source
- No build step, no dependencies, no framework
